HTML
  • HTML is the building block of websites. It handles the content, sctucture, and the element on the page.
  • The head element contains infromation about the webpage.
  • The body element represents the visible conent shown to the user.
  • The 'src' attribute defines the location, or source, of the image file
  • The 'alt' attribute contains a text string that describes the appearance and functionality of the image. This is important for assistive technologies.
  • The footer element appears at the bottom of the page and usually contains the author, copyright, contact, sitemap, and navigation.

CSS
  • The language used to describe the presentation of a website created HTML and how each element on a webpage is rendered.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • CSS selectors are used to find or selecting the HTML elements you want to style.
  • CSS text formatting properties are used to format and style text by setting their color, alignment, spacing, etc. as per requirement.
  • The CSS background properties are used to design the background and define the background effects for elements.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

JavaScript
  • JavaScript is a lightweight programming language used to provide logic and interactivity for websites.
  • JavaScript and Java are not the same!
  • Java is an OOP programming language while Java Script is an OOP scripting language. Java creates applications that run in a virtual machine or browser while JavaScript code is run on a browser only. Java code needs to be compiled while JavaScript code are all in text.
  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
